# Ferlay — Flutter App

Mobile remote control for Claude Code sessions. Start, manage, and approve AI coding agent sessions from your phone — anywhere, anytime.

## Getting Started

### Prerequisites

- Flutter SDK ^3.11.1
- A running [Ferlay daemon](../daemon/) on your dev machine

### Run

```bash
cd app
flutter run
```

### Pair with your daemon

1. Run `ferlay pair` on your dev machine
2. Open the app and scan the QR code (or enter the code manually)
3. Once paired, you can manage sessions remotely
