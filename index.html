Ferlay

Your AI agent, always within reach.

Remote control for Claude Code from your phone. Start, manage, and approve AI coding sessions from anywhere. Install the daemon, scan a QR code, and you're in control. All communication is end-to-end encrypted.

Install

# Linux / macOS
curl -sSL https://raw.githubusercontent.com/y0sif/ferlay/main/scripts/install.sh | sh
# Windows (PowerShell)
irm https://raw.githubusercontent.com/y0sif/ferlay/main/scripts/install.ps1 | iex

Or build from source: cargo install --path daemon — all install methods

📱

Remote Control

Start and manage Claude Code sessions from your phone. No SSH, no VPN, no port forwarding.

🔒

End-to-End Encrypted

X25519 key exchange + AES-256-GCM. The relay server never sees your data.

🖥️

Cross-Platform

Daemon runs on Linux, macOS, and Windows. App available for Android and iOS.

How It Works

Phone App ←→ Relay Server ←→ Daemon ←→ Claude Code

Pair once with a QR code. All messages are encrypted end-to-end.
The relay only forwards opaque ciphertext between your devices.

Get Started

Install the daemon, grab the app, and start coding from anywhere.
